G≈ç potential for you Martini3 simulations.

```
usage: olives.py [-h] [-f F] [--molname MOLNAME] [--contact_map CONTACT_MAP] [--vsdefinition VSDEFINITION] [--vstable VSTABLE] [--vsharm VSHARM] [--exclusion EXCLUSION]

optional arguments:
  -h, --help            show this help message and exit
  -f F                  Path to CG structure generated by martinize2 using the -govs-include -govs-moltype flags
  --molname MOLNAME     Molecule name used as prefix in your output file names and the virtual bead names (default: molecule_0). If you will combine your Go-like model with a
                        coarse-grained protein generated with martinize2, you must use the same name as specified with the --govs-moltype flag of martinize2!
  --contact_map CONTACT_MAP
                        Contact map file calculated by http://pomalab.ippt.pan.pl/GoContactMap/ using the AA structure (default="go_contacts.txt")
  --vsdefinition VSDEFINITION
                        File name suffix used for the .itp file containing the virtual site particles definitions (default="BB-part-def_VirtGoSites")
  --vstable VSTABLE     File name suffix used for the .itp file containing the virtual site particles definitions (default="BB-part-def_VirtGoSites")
  --vsharm VSHARM       File name suffix containing the GO contacts defined as harmonic bonds. Useful for visualizations.
  --exclusion EXCLUSION
                        File name suffix for .itp containing inteaction exlusion between BB atoms already treated with GO potentials.
 ```
